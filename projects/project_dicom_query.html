DICOM
Extract Data from PACS

The Radiology Information System (RIS) is the first entry point to a medical database. Usually, a medical RIS is a SQL database.

With SQL query and filter, I’ll get the type of medical study I’m interested in.

The link between RIS and PACS is the AccessionNumber (AN).

Note: You should avoid querying the PACS with the Study Instance UID (SIUID) from the RIS. The mapping between the RIS SUID and PACS is rarely functional.

Now I will query the PACS with AN

The main idea is to first query the PACS with the AN to get all the SUIDs associated. Then query the PACS with these SUIDs to get the DICOM.

C_FIND code: get SIUDs from AN

C-MOVE: get DICOM from SIUID

That’s the main concept to collect DICOM on a PACS server.
